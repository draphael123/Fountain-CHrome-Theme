# Theme Images

This directory should contain the following image files for the Chrome theme:

- `theme_frame.png` - Main frame image (3400x200px recommended)
- `theme_frame_inactive.png` - Inactive frame image (3400x200px recommended)
- `theme_frame_incognito.png` - Incognito frame image (3400x200px recommended)
- `theme_frame_incognito_inactive.png` - Incognito inactive frame image (3400x200px recommended)
- `theme_toolbar.png` - Toolbar background (3400x150px recommended)
- `theme_tab_background.png` - Tab background (3400x200px recommended)
- `theme_ntp_background.png` - New Tab Page background (1920x1200px recommended)
- `theme_window_control_background.png` - Window control background

For now, the theme uses colors only. You can add custom images here to enhance the theme.
